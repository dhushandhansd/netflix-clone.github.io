# Netflix Clone

Netflix India Cloned with all Original Assets and Validation

## Features

- Language Selection Markdown
- User Authentication
- Unaccessable Home Page for Unauthorized Users
- User Session lockdown with Cookies

## Tech Used

- HTML
- CSS
- JS
